# 📑 Registro Integral Pro (PWA)

Sistema profesional de registro integral optimizado para dispositivos móviles. Diseñado con una interfaz **SaaS Premium**, efectos de glassmorphism y despliegue automático como aplicación nativa.

## 🚀 Acceso a la App
Puedes acceder e instalar la aplicación desde el siguiente enlace:
👉 **[CLIC AQUÍ PARA ABRIR LA APP](https://tu-usuario.github.io/nombre-de-tu-repo/)**

## ✨ Características
* **Instalable:** Funciona como una App nativa (PWA).
* **Modo Offline:** Acceso básico sin conexión mediante Service Workers.
* **Interfaz Pro:** Diseño oscuro con efectos neón y optimización para Notch.
* **Envío Directo:** Integración fluida con WhatsApp.

## 📱 Instalación
1. Abre el enlace en **Chrome** (Android) o **Safari** (iOS).
2. Haz clic en el botón **"Instalar App"** o selecciona **"Agregar a la pantalla de inicio"** en el menú del navegador.
3. ¡Listo! La app aparecerá en tu menú de aplicaciones.
